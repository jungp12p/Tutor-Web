# Tutor Website

This is a simple website for students to access class materials.
The site has two main functions:

1. **Watch recorded videos** (hosted on Google Drive)
2. **Join a Zoom session** with one click

The website is built with basic HTML, CSS, and JavaScript and is deployed using Netlify.

---

## Features

- Sidebar list of video recordings
- Embedded video player
- One-click Zoom join button
- Automatically updates on every GitHub push

---

## How to Update the Website

To apply any changes:

```bash
git add .
git commit -m "Update website"
git push
```
Netlify will automatically rebuild and publish the new version.
